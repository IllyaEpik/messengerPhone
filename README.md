# Messenger

**Brief description:**
In this project we create messenger for phone



## Technologies used
- React Native
- TypeScript



## How start the project

1. Clone the repository:

```git clone <repository_URL> cd messengerPhone```

2. Install the required modules:
```npx expo insrall```

3. Run the project:

```npm run start ```
